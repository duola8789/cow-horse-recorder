# 微信公众平台配置说明

## 节假日 API 域名白名单配置

**必须在发布前完成此配置，否则节假日同步功能无法使用！**

### 配置步骤：

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 进入小程序管理后台
3. 点击左侧菜单：**开发** → **开发管理** → **开发设置**
4. 找到 **服务器域名** 部分
5. 点击 **request 合法域名** 右侧的 **修改** 按钮
6. 添加域名：`https://timor.tech`
7. 点击 **保存并提交**

### 验证配置：

配置完成后，可以在微信开发者工具中测试节假日同步功能：
- 清除小程序数据
- 重启小程序
- 查看控制台是否有 "[HolidayService] 2026 年节假日数据同步成功" 日志

### 注意事项：

- 配置后可能需要几分钟生效
- 如果测试环境需要跳过域名校验，可以在开发者工具中：**详情** → **本地设置** → 勾选 **"不校验合法域名"**
- 但正式发布前必须完成域名配置
